demarcate.js

demarcate.js is a javascript based editor for Markdown. Its only dependency (since v2.0 is showdown.js for Markdown to HTML conversion) It lets you edit directly in a page and generate Markdown back from the HTML elements. demarcate.js is under development by William Hart. It supports most of the basic Markdown tags. The library is offered under the MIT license
